## рҹҡҖ Getting Started

### Prerequisites

Make sure you have the following installed:

- Node.js (v18 or later)
- npm or yarn

### Clone the Repository

```bash
git clone https://github.com/Oywon/Livewave.git
cd Livewave
```

### Install Dependencies

```bash
npm install
```

### Run Development Server

```bash
npm run dev
```

The application will start on your local machine and can be accessed through your browser.
